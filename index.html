Moje skilly

HTML 5

HTML 5

je verze značkovacího jazyka HTML, který se používá k tvorbě webových stránek. Oproti předchozí verzi HTML 4 z roku 1997 přináší podstatné změny.

CSS 3

CSS 3

je už třetí verzí kaskádových stylů. Většina vlastností CSS 3 je podporována nejběžnějšími webovými prohlížeči. CSS 3 přináší např. zaoblené rohy, stíny u textu či obrázků.

JavaScript

JavaScript

je objektově orientovaný, událostmi řízený scriptovací jazyk, jehož autorem je Brendan Eich. Program v JavaScriptu se spouští až po stažení webové stránky z Internetu.

INFO O MNĚ

Umím testovat software jak manuálně, tak i automaticky. Prošla jsem kurzy od Radka Kitnera. Dokážu si vytvořit sama testcase, stavový diagram. Dále jsme se učili testovat pomocí programů JIRA, POSTMAN, TESTRAIL. Pro automatické testování softwaru jsme používali Robot Framework. Také jsem prošla jednodenním kurzem u Czechitas, který byl zaměřený na manuální testování software.

Kóduji webové stránky pomocí CSS, HTML a JS. Weby jsou responzivní. Kód je validní a sémanticky správný. Umím používat devtools, příkazový řádek a učím se JavaScript a Python. Neustále se zlepšuji v HTML 5 a CSS 3.

Manuální Testování

Tester prověřuje program tvořený programátorem a ostatními členy skupiny, zda funguje dle požadavků zadavatele. Testuje se v průběhu vytváření softwaru, což přispívá k rychlejšímu dokončení programu. U manuálního testování musíme ručně prověřit, jestli je výsledek testování validní.

Automatické Testování

Automatické testování softwaru spočívá ve vytvoření kódu, který dokáže opakovaně otestovat danou část programu. Rozdíl mezi automatickým testováním oproti manuálnímu je ten, jakým způsobem budeme kontrolovat výsledek. Automatický test zjistí sám, jestli prošel, či nikoliv. A také zjistí, kde je v programu chyba.
